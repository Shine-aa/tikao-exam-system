# Excel 导入格式详细说明

## 概述

本文档详细说明题库导入功能的 Excel 文件格式要求，包括数据结构、字段规范、数据类型等详细技术规范。

## 文件基本信息

### 支持的文件格式
- **Excel 2007+**: `.xlsx` (推荐)
- **Excel 97-2003**: `.xls`
- **文件大小限制**: 最大 10MB
- **编码格式**: UTF-8

### 工作表要求
- 使用第一个工作表 (Sheet1)
- 第一行必须是表头
- 数据从第二行开始
- 空行会被自动跳过

## 数据结构规范

### 表头定义
```csv
题目类型,题目内容,选项A,选项B,选项C,选项D,选项E,正确答案,难度,分值,知识点,标签,答案解析,图片路径
```

### 字段详细说明

#### 1. 题目类型 (第1列)
- **字段名**: 题目类型
- **数据类型**: 枚举字符串
- **是否必填**: 是
- **允许值**:
  - `SINGLE_CHOICE` - 单选题
  - `MULTIPLE_CHOICE` - 多选题
  - `TRUE_FALSE` - 判断题
  - `FILL_BLANK` - 填空题
  - `SUBJECTIVE` - 主观题
- **示例**: `SINGLE_CHOICE`

#### 2. 题目内容 (第2列)
- **字段名**: 题目内容
- **数据类型**: 文本
- **是否必填**: 是
- **长度限制**: 最大 5000 字符
- **特殊字符**: 支持换行符、特殊符号
- **示例**: `Java中用于继承的关键字是?`

#### 3. 选项A (第3列)
- **字段名**: 选项A
- **数据类型**: 文本
- **是否必填**: 选择题必填
- **长度限制**: 最大 1000 字符
- **示例**: `extends`

#### 4. 选项B (第4列)
- **字段名**: 选项B
- **数据类型**: 文本
- **是否必填**: 选择题必填
- **长度限制**: 最大 1000 字符
- **示例**: `implements`

#### 5. 选项C (第5列)
- **字段名**: 选项C
- **数据类型**: 文本
- **是否必填**: 选择题必填
- **长度限制**: 最大 1000 字符
- **示例**: `inherits`

#### 6. 选项D (第6列)
- **字段名**: 选项D
- **数据类型**: 文本
- **是否必填**: 选择题必填
- **长度限制**: 最大 1000 字符
- **示例**: `super`

#### 7. 选项E (第7列)
- **字段名**: 选项E
- **数据类型**: 文本
- **是否必填**: 否
- **长度限制**: 最大 1000 字符
- **使用场景**: 多选题可选
- **示例**: `double`

#### 8. 正确答案 (第8列)
- **字段名**: 正确答案
- **数据类型**: 文本
- **是否必填**: 是
- **长度限制**: 最大 2000 字符
- **格式要求**:
  - 单选题/判断题: 直接填写选项内容
  - 多选题: 多个答案用分号(;)分隔
  - 填空题/主观题: 填写答案内容
- **示例**: 
  - 单选题: `extends`
  - 多选题: `int;boolean;double`
  - 判断题: `正确`
  - 填空题: `b²-4ac`

#### 9. 难度 (第9列)
- **字段名**: 难度
- **数据类型**: 枚举字符串
- **是否必填**: 否
- **默认值**: `MEDIUM`
- **允许值**:
  - `EASY` - 简单
  - `MEDIUM` - 中等
  - `HARD` - 困难
- **示例**: `EASY`

#### 10. 分值 (第10列)
- **字段名**: 分值
- **数据类型**: 整数
- **是否必填**: 否
- **默认值**: `1`
- **取值范围**: 1-100
- **示例**: `5`

#### 11. 知识点 (第11列)
- **字段名**: 知识点
- **数据类型**: 文本
- **是否必填**: 否
- **长度限制**: 最大 200 字符
- **示例**: `Java基础`

#### 12. 标签 (第12列)
- **字段名**: 标签
- **数据类型**: 文本
- **是否必填**: 否
- **长度限制**: 最大 500 字符
- **格式要求**: 多个标签用竖线(|)分隔
- **示例**: `Java|继承|基础题`

#### 13. 答案解析 (第13列)
- **字段名**: 答案解析
- **数据类型**: 文本
- **是否必填**: 否
- **长度限制**: 最大 5000 字符
- **示例**: `Java使用extends关键字实现继承`

#### 14. 图片路径 (第14列)
- **字段名**: 图片路径
- **数据类型**: 文本
- **是否必填**: 否
- **长度限制**: 最大 500 字符
- **格式要求**: 相对路径或绝对路径
- **示例**: `/uploads/questions/java.jpg`

## 题目类型详细规范

### 单选题 (SINGLE_CHOICE)
```csv
SINGLE_CHOICE,Java中用于继承的关键字是?,extends,implements,inherits,super,,extends,EASY,3,Java基础,Java|继承,Java使用extends关键字实现继承,
```

**要求**:
- 必须填写选项A、B、C、D
- 正确答案填写正确选项的内容
- 选项E可以为空

### 多选题 (MULTIPLE_CHOICE)
```csv
MULTIPLE_CHOICE,以下哪些是Java的基本数据类型?,int,String,boolean,Double,double,int;boolean;double,MEDIUM,5,Java基础,数据类型,String和Double是包装类，不是基本类型,
```

**要求**:
- 必须填写选项A、B、C、D
- 选项E可选
- 正确答案用分号(;)分隔多个选项内容

### 判断题 (TRUE_FALSE)
```csv
TRUE_FALSE,Java中String类是不可变的,正确,错误,,,,正确,EASY,3,Java基础,String,String对象一旦创建就不能被修改,
```

**要求**:
- 选项A填写"正确"
- 选项B填写"错误"
- 选项C、D、E可以为空
- 正确答案填写"正确"或"错误"

### 填空题 (FILL_BLANK)
```csv
FILL_BLANK,一元二次方程ax²+bx+c=0的判别式Δ=______,,,,,,b²-4ac,MEDIUM,5,数学,代数|方程,判别式用于判断方程的根的情况,
```

**要求**:
- 选项列可以为空
- 正确答案填写填空的答案

### 主观题 (SUBJECTIVE)
```csv
SUBJECTIVE,求函数f(x)=x²-4x+3的极值点和极值,,,,,,对f(x)求导得f'(x)=2x-4，令f'(x)=0得x=2。将x=2代入f(x)得f(2)=-1。二阶导数f''(x)=2>0，所以x=2是极小值点，极小值为-1。,HARD,10,数学,微积分|极值,通过求导和二阶导数判断极值,
```

**要求**:
- 选项列可以为空
- 正确答案填写参考答案

## 数据验证规则

### 必填字段验证
- 题目类型: 不能为空
- 题目内容: 不能为空
- 正确答案: 不能为空
- 选择题选项: 至少填写选项A、B、C、D

### 格式验证
- 题目类型: 必须是有效的枚举值
- 难度: 必须是 EASY、MEDIUM、HARD 之一
- 分值: 必须是 1-100 之间的整数
- 多选题答案: 必须用分号(;)分隔

### 长度验证
- 题目内容: 最大 5000 字符
- 选项内容: 最大 1000 字符
- 正确答案: 最大 2000 字符
- 答案解析: 最大 5000 字符

## 错误处理

### 常见错误类型
1. **格式错误**: 题目类型、难度等枚举值不正确
2. **必填字段缺失**: 题目内容、正确答案等必填字段为空
3. **长度超限**: 文本字段超过最大长度限制
4. **数据类型错误**: 分值不是数字等

### 错误信息格式
```
第 {行号} 行: {错误描述}
```

**示例**:
```
第 3 行: 题目类型不能为空
第 5 行: 无效的题目类型: INVALID_TYPE
第 7 行: 选择题必须至少有一个选项
```
