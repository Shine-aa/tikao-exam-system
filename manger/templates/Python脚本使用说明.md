# Python 脚本使用说明

## 文件说明

`generate_test_excel.py` 是一个用于生成题库导入测试 Excel 文件的 Python 脚本。

## 功能

该脚本可以生成包含 31 道测试题目的 Excel 文件，包括：
- 22 道常规题目（单选题、多选题、判断题、填空题、主观题）
- 9 道程序题（Java、Python、C++）

## 环境要求

### 1. Python 版本
- Python 3.6 或更高版本

### 2. 依赖库
需要安装 `openpyxl` 库：

```bash
pip install openpyxl
```

或者使用 pip3：

```bash
pip3 install openpyxl
```

## 运行方法

### Windows PowerShell

1. **打开 PowerShell 或终端**

2. **进入脚本所在目录**
   ```powershell
   cd C:\Users\用户名\Desktop\zonghexiangmu\manger\templates
   ```

3. **运行脚本**
   ```powershell
   python generate_test_excel.py
   ```
   
   或者：
   ```powershell
   python3 generate_test_excel.py
   ```

### Windows CMD

```cmd
cd C:\Users\卢本伟\Desktop\zonghexiangmu\manger\templates

```

### 运行结果

运行成功后，会在当前目录生成 `题库导入测试.xlsx` 文件，并显示以下信息：

## 程序题代码示例

### Java 程序题示例

**题目内容：**
```
编写一个Java程序，计算两个整数的和并输出结果。程序需要从标准输入读取两个整数，计算它们的和，然后输出结果。
要求：
1. 使用Scanner类读取输入
2. 计算两数之和
3. 输出结果
```

**参考答案代码：**
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int a = scanner.nextInt();
        int b = scanner.nextInt();
        int sum = a + b;
        System.out.println(sum);
        scanner.close();
    }
}
```

### Python 程序题示例

**题目内容：**
```
编写一个Python程序，实现冒泡排序算法，对整数数组进行升序排序。
要求：
1. 方法签名：def bubble_sort(arr)
2. 原地排序，修改原数组
3. 使用冒泡排序算法
4. 从标准输入读取整数（用空格分隔），排序后输出
```

**参考答案代码：**
```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]

if __name__ == "__main__":
    arr = list(map(int, input().split()))
    bubble_sort(arr)
    print(" ".join(map(str, arr)))
```

### C++ 程序题示例

**题目内容：**
```
编写一个C++程序，计算斐波那契数列的第n项。
要求：
1. 使用递归或循环实现
2. 从标准输入读取一个整数n（n >= 0）
3. 输出斐波那契数列的第n项
4. 斐波那契数列定义：F(0)=0, F(1)=1, F(n)=F(n-1)+F(n-2)
```

**参考答案代码：**
```cpp
#include <iostream>
using namespace std;

int fibonacci(int n) {
    if (n <= 1) return n;
    int a = 0, b = 1, c;
    for (int i = 2; i <= n; i++) {
        c = a + b;
        a = b;
        b = c;
    }
    return b;
}

int main() {
    int n;
    cin >> n;
    cout << fibonacci(n) << endl;
    return 0;
}
```

## 注意事项

### 1. 多选题答案格式
- 多选题的正确答案必须用**分号（;）**分隔
- 例如：`选项A;选项B;选项C`

### 2. 标签格式
- 多个标签用**竖线（|）**分隔
- 例如：`Java|继承|面向对象`

### 3. 程序题注意事项
- 程序题的代码必须完整，包含必要的导入语句和主函数
- Java 程序的主类名必须是 `Main`
- Python 程序需要包含 `if __name__ == "__main__":` 块
- C++ 程序需要包含 `main` 函数
- **测试用例格式**：`输入|输出;输入|输出`（用`|`分隔输入和输出，用`;`分隔多个测试用例）
- 多行输入使用 `\n` 表示换行，例如：`9\n30|9和30之间有6个素数`
- 测试用例是可选的，但建议为程序题提供至少2-3个测试用例

### 4. 填空题填空位置
- 填空题的题目内容中用 `______` 表示填空位置
- 例如：`一元二次方程ax²+bx+c=0的判别式Δ=______`

### 5. 必填字段
- **题目类型**：必填
- **题目内容**：必填
- **正确答案**：必填
- **选择题的选项**：选择题必填

### 6. 可选字段
- 难度、分值、知识点、标签、答案解析、图片路径、编程语言（非程序题）、测试用例（非程序题）都是可选的
- 如果不填写，留空即可
- 程序题的测试用例建议填写，用于自动判题

## 常见问题

### Q1: 运行脚本时提示 "ModuleNotFoundError: No module named 'openpyxl'"
**A:** 需要先安装 openpyxl 库：
```bash
pip install openpyxl
```

### Q2: 生成的 Excel 文件在哪里？
**A:** 文件会生成在运行脚本的当前目录下，文件名为 `题库导入测试.xlsx`

### Q3: 如何修改生成的文件名？
**A:** 修改脚本中的 `output_file` 变量：
```python
output_file = '你的文件名.xlsx'
```

### Q4: 程序题的代码格式有要求吗？
**A:** 是的，代码必须完整可运行。Java 的主类必须是 `Main`，Python 需要包含主程序入口，C++ 需要包含 `main` 函数。

### Q6: 程序题的测试用例格式是什么？
**A:** 测试用例格式为：`输入|输出;输入|输出`。用`|`分隔输入和输出，用`;`分隔多个测试用例。多行输入使用`\n`表示换行。例如：`7|7以内所有素数之和为10;2|2以内所有素数之和为0`

### Q5: 多选题答案格式错误怎么办？
**A:** 确保多选题的答案用分号（;）分隔，例如：`选项A;选项B;选项C`
