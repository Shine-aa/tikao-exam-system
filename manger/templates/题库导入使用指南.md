# 题库导入使用指南

## 概述

本系统支持通过 Excel 文件批量导入题目到题库中。本文档详细说明如何准备和导入题目数据。

## 文件说明

### 1. Python 脚本文件
- **文件名**: `generate_test_excel.py`
- **作用**: 生成符合系统要求的测试 Excel 文件
- **位置**: `manger/templates/generate_test_excel.py`

### 2. Excel 文件
- **文件名**: `题库导入测试.xlsx`
- **作用**: 包含测试题目的 Excel 文件
- **位置**: `manger/templates/题库导入测试.xlsx`

## Excel 文件格式规范

### 表头结构
Excel 文件必须包含以下 14 列，按顺序排列：

| 列号 | 列名 | 说明 | 是否必填 | 示例 |
|------|------|------|----------|------|
| 1 | 题目类型 | 题目的类型 | 是 | SINGLE_CHOICE |
| 2 | 题目内容 | 题目的具体内容 | 是 | Java中用于继承的关键字是? |
| 3 | 选项A | 第一个选项内容 | 选择题必填 | extends |
| 4 | 选项B | 第二个选项内容 | 选择题必填 | implements |
| 5 | 选项C | 第三个选项内容 | 选择题必填 | inherits |
| 6 | 选项D | 第四个选项内容 | 选择题必填 | super |
| 7 | 选项E | 第五个选项内容 | 多选题可选 | double |
| 8 | 正确答案 | 正确答案内容 | 是 | extends |
| 9 | 难度 | 题目难度等级 | 否 | EASY |
| 10 | 分值 | 题目分值 | 否 | 3 |
| 11 | 知识点 | 题目所属知识点 | 否 | Java基础 |
| 12 | 标签 | 题目标签 | 否 | Java\|继承 |
| 13 | 答案解析 | 题目解析说明 | 否 | Java使用extends关键字实现继承 |
| 14 | 图片路径 | 题目图片路径 | 否 | /uploads/question.jpg |

### 题目类型说明

#### 1. SINGLE_CHOICE (单选题)
- **选项要求**: 至少填写选项A、B、C、D
- **正确答案**: 填写正确选项的内容
- **示例**:
  ```
  题目类型: SINGLE_CHOICE
  题目内容: Java中用于继承的关键字是?
  选项A: extends
  选项B: implements
  选项C: inherits
  选项D: super
  正确答案: extends
  ```

#### 2. MULTIPLE_CHOICE (多选题)
- **选项要求**: 至少填写选项A、B、C、D，可选择性填写选项E
- **正确答案**: 多个正确选项内容用分号(;)分隔
- **示例**:
  ```
  题目类型: MULTIPLE_CHOICE
  题目内容: 以下哪些是Java的基本数据类型?
  选项A: int
  选项B: String
  选项C: boolean
  选项D: Double
  选项E: double
  正确答案: int;boolean;double
  ```

#### 3. TRUE_FALSE (判断题)
- **选项要求**: 选项A填写"正确"，选项B填写"错误"
- **正确答案**: 填写"正确"或"错误"
- **示例**:
  ```
  题目类型: TRUE_FALSE
  题目内容: Java中String类是不可变的
  选项A: 正确
  选项B: 错误
  正确答案: 正确
  ```

#### 4. FILL_BLANK (填空题)
- **选项要求**: 不需要填写选项
- **正确答案**: 填写填空的答案
- **示例**:
  ```
  题目类型: FILL_BLANK
  题目内容: 一元二次方程ax²+bx+c=0的判别式Δ=______
  正确答案: b²-4ac
  ```

#### 5. SUBJECTIVE (主观题)
- **选项要求**: 不需要填写选项
- **正确答案**: 填写参考答案
- **示例**:
  ```
  题目类型: SUBJECTIVE
  题目内容: 求函数f(x)=x²-4x+3的极值点和极值
  正确答案: 对f(x)求导得f'(x)=2x-4，令f'(x)=0得x=2。将x=2代入f(x)得f(2)=-1。二阶导数f''(x)=2>0，所以x=2是极小值点，极小值为-1。
  ```

### 难度等级说明
- **EASY**: 简单
- **MEDIUM**: 中等
- **HARD**: 困难

### 标签格式
- 多个标签用竖线(|)分隔
- 示例: `Java|继承|基础题`

## 使用步骤

### 1. 生成测试文件
```bash
# 进入模板目录
cd manger/templates

# 运行Python脚本生成Excel文件
python generate_test_excel.py
```

### 2. 导入题目
1. 登录系统，进入教师端
2. 访问"题库管理"页面 (`http://localhost:3000/teacher/question-bank`)
3. 点击"导入题库"按钮
4. 选择生成的 `题库导入测试.xlsx` 文件
5. 点击"开始导入"
6. 查看导入结果

### 3. 自定义题目
如需导入自定义题目，请按照以下步骤：

1. **复制模板**: 复制 `题库导入测试.xlsx` 文件
2. **编辑内容**: 按照格式规范填写题目信息
3. **保存文件**: 确保文件格式为 `.xlsx` 或 `.xls`
4. **导入系统**: 按照上述导入步骤操作

## 注意事项

### 1. 文件格式要求
- 文件必须是 Excel 格式 (`.xlsx` 或 `.xls`)
- 第一行必须是表头
- 数据从第二行开始填写

### 2. 必填字段
- 题目类型
- 题目内容
- 正确答案
- 选择题的选项内容

### 3. 数据格式
- 多选题正确答案用分号(;)分隔
- 标签用竖线(|)分隔
- 难度等级必须是指定的枚举值
- 分值必须是数字

### 4. 常见错误
- **题目类型错误**: 确保使用正确的枚举值
- **选项缺失**: 选择题必须填写选项内容
- **答案格式错误**: 多选题答案用分号分隔
- **难度等级错误**: 使用 EASY、MEDIUM、HARD
