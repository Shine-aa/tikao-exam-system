# 题目导入格式说明

## 系统概述

本系统支持通过 Excel 文件批量导入题目，支持多种题目类型，包括单选题、多选题、判断题、填空题和主观题。

## 导入功能特点

- ✅ 支持 5 种题目类型
- ✅ 批量导入，提高效率
- ✅ 数据验证，确保质量
- ✅ 错误提示，便于修正
- ✅ 进度显示，用户体验好

## Excel 文件要求

### 基本要求
- **文件格式**: `.xlsx` 或 `.xls`
- **文件大小**: 最大 10MB
- **工作表**: 使用第一个工作表
- **编码**: UTF-8

### 表头结构
Excel 文件必须包含以下 14 列，顺序不能改变：

```
题目类型 | 题目内容 | 选项A | 选项B | 选项C | 选项D | 选项E | 正确答案 | 难度 | 分值 | 知识点 | 标签 | 答案解析 | 图片路径
```

## 题目类型详解

### 1. 单选题 (SINGLE_CHOICE)

**特点**: 只有一个正确答案的选择题

**格式要求**:
- 必须填写选项A、B、C、D
- 选项E可选
- 正确答案填写正确选项的内容

**示例**:
```
题目类型: SINGLE_CHOICE
题目内容: Java中用于继承的关键字是?
选项A: extends
选项B: implements  
选项C: inherits
选项D: super
选项E: (空)
正确答案: extends
```

**注意事项**:
- 选项内容要简洁明了
- 避免选项内容过长
- 确保只有一个正确答案

### 2. 多选题 (MULTIPLE_CHOICE)

**特点**: 有多个正确答案的选择题

**格式要求**:
- 必须填写选项A、B、C、D
- 选项E可选
- 正确答案用分号(;)分隔多个选项内容

**示例**:
```
题目类型: MULTIPLE_CHOICE
题目内容: 以下哪些是Java的基本数据类型?
选项A: int
选项B: String
选项C: boolean
选项D: Double
选项E: double
正确答案: int;boolean;double
```

**注意事项**:
- 正确答案必须用分号(;)分隔
- 不要用逗号或其他符号分隔
- 确保答案内容与选项内容完全一致

### 3. 判断题 (TRUE_FALSE)

**特点**: 判断题目陈述是否正确

**格式要求**:
- 选项A填写"正确"
- 选项B填写"错误"
- 选项C、D、E可以为空
- 正确答案填写"正确"或"错误"

**示例**:
```
题目类型: TRUE_FALSE
题目内容: Java中String类是不可变的
选项A: 正确
选项B: 错误
选项C: (空)
选项D: (空)
选项E: (空)
正确答案: 正确
```

**注意事项**:
- 选项A必须是"正确"
- 选项B必须是"错误"
- 题目陈述要明确，避免歧义

### 4. 填空题 (FILL_BLANK)

**特点**: 在空白处填入正确答案

**格式要求**:
- 选项列可以为空
- 正确答案填写填空的答案
- 题目中用下划线(____)表示空白

**示例**:
```
题目类型: FILL_BLANK
题目内容: 一元二次方程ax²+bx+c=0的判别式Δ=______
选项A: (空)
选项B: (空)
选项C: (空)
选项D: (空)
选项E: (空)
正确答案: b²-4ac
```

**注意事项**:
- 答案要准确，避免拼写错误
- 数学公式要正确
- 考虑答案的多种表达方式

### 5. 主观题 (SUBJECTIVE)

**特点**: 需要详细回答的开放性题目

**格式要求**:
- 选项列可以为空
- 正确答案填写参考答案
- 支持较长的文本内容

**示例**:
```
题目类型: SUBJECTIVE
题目内容: 求函数f(x)=x²-4x+3的极值点和极值
选项A: (空)
选项B: (空)
选项C: (空)
选项D: (空)
选项E: (空)
正确答案: 对f(x)求导得f'(x)=2x-4，令f'(x)=0得x=2。将x=2代入f(x)得f(2)=-1。二阶导数f''(x)=2>0，所以x=2是极小值点，极小值为-1。
```

**注意事项**:
- 参考答案要详细完整
- 包含解题步骤
- 考虑多种解题方法

## 字段说明

### 必填字段

| 字段名 | 说明 | 示例 |
|--------|------|------|
| 题目类型 | 题目的类型 | SINGLE_CHOICE |
| 题目内容 | 题目的具体内容 | Java中用于继承的关键字是? |
| 正确答案 | 正确答案内容 | extends |

### 可选字段

| 字段名 | 说明 | 默认值 | 示例 |
|--------|------|--------|------|
| 难度 | 题目难度等级 | MEDIUM | EASY |
| 分值 | 题目分值 | 1 | 5 |
| 知识点 | 题目所属知识点 | - | Java基础 |
| 标签 | 题目标签 | - | Java\|继承 |
| 答案解析 | 题目解析说明 | - | Java使用extends关键字实现继承 |
| 图片路径 | 题目图片路径 | - | /uploads/question.jpg |

### 难度等级
- **EASY**: 简单 - 基础概念题
- **MEDIUM**: 中等 - 应用理解题  
- **HARD**: 困难 - 综合分析题

### 标签格式
- 多个标签用竖线(|)分隔
- 示例: `Java|继承|基础题`
- 建议使用: 学科|知识点|难度

## 数据验证规则

### 格式验证
- 题目类型必须是有效的枚举值
- 难度必须是 EASY、MEDIUM、HARD 之一
- 分值必须是 1-100 之间的整数

### 内容验证
- 题目内容不能为空
- 选择题必须至少有一个选项
- 正确答案不能为空

### 长度限制
- 题目内容: 最大 5000 字符
- 选项内容: 最大 1000 字符
- 正确答案: 最大 2000 字符
- 答案解析: 最大 5000 字符

## 导入步骤

### 1. 准备文件
1. 下载模板文件 `题库导入测试.xlsx`
2. 按照格式要求填写题目信息
3. 检查数据完整性和正确性
4. 保存为 Excel 格式

### 2. 上传导入
1. 登录系统，进入教师端
2. 访问"题库管理"页面
3. 点击"导入题库"按钮
4. 选择准备好的 Excel 文件
5. 点击"开始导入"

### 3. 查看结果
1. 等待导入完成
2. 查看导入结果统计
3. 如有错误，查看错误详情
4. 修正错误后重新导入

## 常见问题

### Q1: 多选题答案格式错误
**问题**: 多选题答案用逗号分隔
**解决**: 使用分号(;)分隔，如 `A;B;C`

### Q2: 题目类型填写错误
**问题**: 题目类型不是有效的枚举值
**解决**: 使用正确的类型: SINGLE_CHOICE、MULTIPLE_CHOICE、TRUE_FALSE、FILL_BLANK、SUBJECTIVE

### Q3: 选择题选项缺失
**问题**: 单选题或多选题没有填写选项
**解决**: 至少填写选项A、B、C、D

### Q4: 文件格式不支持
**问题**: 上传的文件不是 Excel 格式
**解决**: 确保文件是 .xlsx 或 .xls 格式

### Q5: 文件过大
**问题**: 文件超过 10MB 限制
**解决**: 分批导入或压缩文件内容
